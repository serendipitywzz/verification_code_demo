# 使用Java实现随机字符串验证码

1. 创建画板： new BufferedImage()
2. 创建画笔：画板.getGraphics()
3. 随机生成内容：不同颜色的字符
4. 绘制内容：画笔.drawString()
5. 存为图片发送: ImageIO.write(画板,图片类型,输出流)

# 算术验证码与随机字符串验证码的区别

算术验证码与随机字符串相比，区别在于，需要用户对显示的表达式进行计算，将结果填写文本框中

2. 新建页面，按格式调用验证码生成代码
3. 自行扩展服务器端代码
